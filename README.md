# Airbnb-Group-Case-Study
🏡 Airbnb Data Analysis Case Study (NYC)
📌 Overview

This project delivers an end-to-end exploratory data analysis (EDA) of Airbnb listings in New York City, focusing on uncovering actionable insights related to pricing, availability, and market distribution.

It demonstrates core data analyst / data science skills including data cleaning, feature understanding, statistical exploration, and visualization to support business decision-making.

🎯 Business Problem

How can Airbnb hosts optimize pricing and availability to maximize occupancy and revenue?

🔍 Key Contributions
Cleaned and processed ~48K records with missing values and inconsistent data
Performed EDA across 15+ features to identify pricing drivers
Built visualizations to highlight location, room type, and demand trends
Translated data findings into business-focused insights
🧰 Tech Stack
Python: Pandas, NumPy
Visualization: Matplotlib, Seaborn
Environment: Jupyter Notebook
📊 Key Insights
Market Concentration: Manhattan (~44%) and Brooklyn (~41%) dominate listings
Pricing Drivers: Entire homes/apartments significantly outperform shared/private rooms
Demand Patterns: Listings with higher reviews show stronger booking activity
Availability Trends: Avg. ~112 days/year indicates seasonal or selective renting behavior
